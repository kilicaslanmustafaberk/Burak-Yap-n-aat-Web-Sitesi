# Burak Yapı İnşaat

Basit kurumsal web sitesi — proje kökünde bulunan `index.html` ile çalışır.

## İçerik
- `index.html` — Ana sayfa
- `img/` — Görseller

## Çalıştırma
1. Proje klasörünü açın.
2. Bir tarayıcıda `index.html` dosyasını açın (çift tıklayarak veya tarayıcıya sürükleyerek).
